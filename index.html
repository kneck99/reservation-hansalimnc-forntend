예약 전용 프론트

연수원 · 회의실 예약

프론트는 GitHub Pages, API는 Cloudflare Worker, 내부 자동화는 Google Apps Script로 분리한 추천 구조입니다. 결제 검증은 Worker에서 처리하고, 최종 예약 저장·캘린더 등록·메일 발송은 Apps Script가 담당합니다.

연수원 이용요금

  • 1~8인: 150,000원
  • 9인부터 1인당 20,000원 추가

예약 정보 입력

예상 결제금액
150,000원

구조 안내

  1. GitHub Pages가 예약 화면 제공
  2. Cloudflare Worker가 금액 계산·예약 검증·결제 검증 처리
  3. Apps Script가 시트·캘린더·메일 자동화 담당

현재 상태

이 패키지는 실결제 시크릿 키를 포함하지 않은 개발용 통합본입니다. PortOne 실연동 시에는 Worker 환경변수와 Apps Script 웹앱 URL을 입력해야 합니다.
